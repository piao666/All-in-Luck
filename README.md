# All in Luck 德州扑克酒吧微信小程序

## 当前版本

v0.1.0-ui-demo

## 当前定位

当前项目是 All in Luck 微信小程序 UI 演示版，用于客户验收页面风格、页面结构和基础操作路径。

## 已完成页面

- 首页
- 点单页
- 购物车页
- 我的页
- 商品详情页
- 积分兑换页
- 会员充值页
- 订单列表页
- 订单详情页

## 当前已支持的演示能力

- 页面跳转
- Mock 数据展示
- 本地购物车缓存演示
- Toast / Modal 演示提示
- 商品加购到购物车
- 订单页面静态展示
- 积分兑换和充值演示

## 当前未接入

- 微信支付

- 导入项目根目录
- 使用测试 AppID 或正式 AppID
- 编译预览
