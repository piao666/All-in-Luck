# All in Luck 微信小程序 UI 演示版项目状态

## 1. 项目定位

当前项目是 All in Luck 德州扑克酒吧微信小程序 UI 演示版。

当前阶段只做：
- 静态 UI
- Mock 数据
- 页面跳转
- Toast / Modal 演示反馈
- 本地购物车缓存演示

当前阶段不做：
- 后端接口
- 数据库
- 真实微信支付
- 真实微信登录
- 手机号认证
- 云开发
- 管理后台

后续所有修改必须遵守：

1. 主背景色：#13164F
2. 强调红：#FF3D45
3. 金色点缀：#D8A84A
4. 浅蓝商品卡：#DCEEFF
5. 页面浅灰背景：#F5F5F5
6. 按钮必须固定 width 和 height
7. 圆形按钮必须 width = height
8. 普通图标不得超过 56rpx
9. 禁止大红色椭圆按钮
10. 禁止巨大悬浮按钮
11. 禁止遮挡内容的全局浮层
12. 禁止恢复“已选 X 件”浮层
13. 禁止通过 padding 或 min-width 把按钮撑大
14. 内容不得被 TabBar 或底部固定栏遮挡
